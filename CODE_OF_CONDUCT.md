# Code of Conduct

## Our Standard

Project spaces should stay respectful, constructive, and focused on improving the software. Harassment, abuse,
discriminatory behavior, threats, and sustained personal attacks are not acceptable.

## Scope

This code of conduct applies to project repositories, issues, pull requests, discussions, release channels, and direct
maintainer interactions related to the project.

## Reporting

Report conduct concerns through the project's GitHub issue tracker when public discussion is appropriate. For private or
security-adjacent reports, contact the maintainer at `jongwan56@gmail.com`.

Maintainers may remove comments, close threads, block users, or otherwise moderate participation when needed to keep the
project usable and safe.
